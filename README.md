# @faustbrian/node-base64

[![npm version](https://badgen.net/npm/v/@faustbrian/node-base64)](https://npm.im/@faustbrian/node-base64)

## Installation

```
pnpm install @faustbrian/node-base64
```

## Usage

```ts
import { encode, decode } from "@faustbrian/node-base64";

decode(encode("a"));
```
